=====================================================
FIVE-STAR JAPAN GROUP — HP 使い方説明書
=====================================================

【フォルダ構成】
five-star-japan/
├── index.html          ← メインページ（触らなくてOK）
├── css/
│   └── style.css       ← デザイン（触らなくてOK）
├── js/
│   ├── contents.js     ← ★★ ここだけ編集すればOK ★★
│   └── main.js         ← 動作（触らなくてOK）
├── images/             ← 画像をここに入れる
└── README.txt          ← この説明書


=====================================================
【文章・情報を更新する方法】
=====================================================
1. 「js/contents.js」をメモ帳（またはテキストエディタ）で開く
2. 変えたい文字を書き換える
   例）電話番号を変えたい場合：
     変更前：tel: '0120-XXX-XXX',
     変更後：tel: '0120-123-456',
3. 保存する（Cmd + S）
4. ブラウザでindex.htmlを開いて確認する

★ シングルクォーテーション（'）の中だけを変えてください
★ 記号や構造は変えないようにしてください


=====================================================
【画像を追加する方法】
=====================================================
1. 使いたい画像ファイルを「images」フォルダに入れる
2. contents.js の img: 'images/ファイル名.jpg' を実際のファイル名に変える

例）カーサービスの最初の画像を変えたい場合：
  img: 'images/car-battery.jpg'  ← この名前で画像を保存すればOK

★ 画像は .jpg または .png 形式が使えます
★ ファイル名は半角英数字推奨（スペースなし）


=====================================================
【ネットに公開する方法（Netlify）】
=====================================================
1. https://netlify.com にアクセス
2. 「Sign up」→ メールアドレスで無料登録
3. ログイン後、ダッシュボードに「five-star-japan」フォルダを
   ドラッグ＆ドロップするだけで公開完了！

★ 独自ドメイン（例：five-star-japan.jp）も設定できます
★ 問い合わせフォームもNetlifyが自動で受け取ってくれます
   → Netlifyのダッシュボード「Forms」で確認できます

======================================================
【問い合わせフォームの受信を確認する場所】
=====================================================
Netlifyにアップ後：
  Netlify管理画面 → あなたのサイト → 「Forms」タブ
  ここに送信された内容が一覧で表示されます。
  メール通知も設定できます（Forms → Form notifications）


=====================================================
【ヒーローの背景画像を変える】
=====================================================
1. 夜の道路などの画像を「images/hero-bg.jpg」として保存
2. 自動的に背景に表示されます

★ 横長の画像（1920×1080以上）がきれいに表示されます
